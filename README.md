verbalizer
==========

Zamiana liczb całkowitych na słownie. Biblioteka zawiera metodę do zamiany liczby całkowitej na jej zapis słowny.

**Sposób użycia**

120.Verbalize(); 

zwraca:

string "sto dwadzieścia"


** na chwilę obecną maksymalna obsługiwana liczba to: 999 999 999 **
